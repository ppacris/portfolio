CGC-FSS | Faculty Scheduling System

CGC-FSS is a scheduling system built for CGC (CITI Global College) that allows the scheduler to automate the scheduling process. Deployed on ph.000webhost.

Technology used:
HTML, CSS (Bootstrap v5), JavaScript, PHP, MySQL, Excel spreadsheet, phpMailer, DataTables, html2canvas, FontAwesome.
